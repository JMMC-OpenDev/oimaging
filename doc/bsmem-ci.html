BSMEM

BSMEM: BiSpectrum Maximum Entropy Method

  • INIT_IMG : initial image

  • MAXITER : maximum number of iterations

  • AUTO_WGT: if set: RGL_WGTis automatically set

  • RGL_WGT : hyper-parameter tuning the strength of the regularization

  • RGL_PRIO: prior image

Reference

Buscher,D. F. “Direct maximum-entropy image reconstruction from the bispectrum”, in J. G. Robertson and W. J. Tango, editors, IAU Symp. 158: Very High Angular Resolution Imaging, p.91 (1994) doi
